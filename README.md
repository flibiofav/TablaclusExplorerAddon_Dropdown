# TablaclusExplorerAddon_Dropdown
Adds "Folder Dropdown" to the type.

Coming from Explorer+QTTabbar, I missed the dropdown funcionality in the "toolbar" addon. Add a new item, select type "Folder dropdown" and choose a folder in the "options" button. On click, shows a dropdown list with the content of the folder; on double-click, opens the folder in a new tab.
